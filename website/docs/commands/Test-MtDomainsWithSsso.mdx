## SYNOPSIS

Check data from Defender for Identity to check domains with Seamless SSO usage.

## SYNTAX

```powershell
Test-MtDomainsWithSsso [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

Executes KQL function over IdentityLogonEvents data to retrieve information about domains with Seamless SSO usage.
It enriches the data with device insights.

## EXAMPLES

### EXAMPLE 1

```powershell
Test-MtDomainsWithSsso
Returns a detailed list of domains with Seamless SSO usage, including associated devices and their properties.
```
